# Obsidian Think

Use a concise thinking loop for non-trivial problems.

## Loop

1. Frame the problem and desired decision.
2. List known facts from the wiki or current context.
3. Identify assumptions and uncertainty.
4. Generate options and tradeoffs.
5. Choose the next action or file an open question.

If the result should persist, save it with `obsidian-save` into `wiki/questions/`.
