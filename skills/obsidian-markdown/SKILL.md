# Obsidian Markdown

Use plain Markdown plus Obsidian wikilinks.

## Rules

- Put YAML frontmatter on maintained wiki pages.
- Link pages with `[[slug|Title]]` when filenames are slugged.
- Use callouts such as `> [!contradiction]` for conflicting claims.
- Keep `.raw/` source material immutable.
- Prefer short atomic notes with a summary and links section.

Do not require community plugins for baseline note rendering.
